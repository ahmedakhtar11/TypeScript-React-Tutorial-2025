# TypeScript React Tutorial 2025

An interactive React application that teaches TypeScript fundamentals through hands-on examples and explanations.

## Features

- **Interactive Tutorial Sections**: Navigate through 4 comprehensive sections covering TypeScript essentials
- **Code Examples**: See real TypeScript code examples with syntax highlighting
- **Progress Tracking**: Visual progress bar shows your current position in the tutorial
- **Difficulty Indicators**: Each section is marked with its difficulty level (Beginner, Intermediate, Advanced)
- **Modern UI**: Dark theme with responsive design

## Tutorial Topics

1. **Type Annotations** - Learn how to define types for variables, function parameters, and return values
2. **Interfaces** - Understand how to define the structure of objects
3. **Generics** - Create reusable components that work with multiple types
4. **Union Types** - Allow values to be one of several types

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/ahmedakhtar11/TypeScript-React-Tutorial-2025.git

# Navigate to the project directory
cd TypeScript-React-Tutorial-2025

# Install dependencies
npm install

# Start the development server
npm start
```

The application will open in your browser at `http://localhost:3000`.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Runs the test suite
- `npm run build` - Builds the app for production
